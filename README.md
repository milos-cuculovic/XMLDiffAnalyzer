# XMLDiffAnalyzer
Python tool for analyzing recent XML Diff algorithm implementations
* xydiff
* jndiff
* diffxml
* jxydiff
* fc-xmldiff
* xcc
* node-delta
* deltaXML
* xmldiff
* xdiff
* xop
* diffmk

In order to have the tool working, the following pip extensions are needed:
* pip install xlsxwriter 
* pip install psutil
